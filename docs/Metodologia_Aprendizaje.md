### **Ejemplo Corregido con Rest/Spread Operators:**

#### **Explicación Teórica:**  

Los operadores **Rest** y **Spread** (`...`) son herramientas clave en JavaScript para trabajar con arrays, objetos y parámetros. Ambos utilizan la misma sintaxis (`...`), pero cumplen diferentes funciones dependiendo del contexto:

---

1. **Rest Operator (`...`)**  
   - Se usa para recolectar múltiples elementos en una sola variable.  
   - Es útil cuando no sabemos cuántos argumentos recibirá una función o cuando necesitamos separar partes de un array/objeto.  

   **Ejemplo Explicativo Completo:**
   ```javascript
   // Uso en una función con argumentos variables
   function sum(...numbers) {
       return numbers.reduce((acc, current) => acc + current, 0);
   }

   console.log(sum(1, 2, 3, 4)); // 10

   // Uso con desestructuración de arrays
   const [first, ...rest] = [1, 2, 3, 4];
   console.log(first); // 1
   console.log(rest);  // [2, 3, 4]

2. **Spread Operator (`...`)**  
   - Se usa para expandir elementos de un array u objeto.  
   - Es ideal para concatenar arrays, clonar objetos o pasar múltiples argumentos a funciones.

   **Ejemplo Explicativo Completo:**
   ```javascript
   // Expansión de un array
      const array1 = [1, 2, 3];
      const array2 = [...array1, 4, 5];
      console.log(array2); // [1, 2, 3, 4, 5]

   // Clonación y expansión en objetos
      const obj1 = { name: "Marcos", age: 25 };
      const obj2 = { ...obj1, location: "Spain" };
      console.log(obj2); 
   // { name: "Marcos", age: 25, location: "Spain" }
